Taller de Introducción a la Investigación y Comunicación Científica

Curso de naturaleza teórico-práctica, desarrollada bajo el modelo COIL (Collaborative Online International Learning), conectando a estudiantes de Perú y Colombia.

Bienvenido estimado docente

Al finalizar el curso-taller, usted comprenderá las fases teóricas de un proceso de investigación científica y podrá aplicar herramientas multimedia y de visualización de datos para comunicar y divulgar conceptos científicos de forma efectiva, en un entorno de colaboración internacional (Perú-Colombia).

PROGRAMACIÓN DE CONTENIDOS

El curso se estructura en tres (3) sesiones síncronas obligatorias (vía Zoom/Meet), complementadas con trabajo autónomo de asesoría en equipos binacionales.

DESCARGA DEL MATERIAL

Puede descargar y organizar recursos los de cada sesión y mantenerlos siempre disponibles.

COMUNIDAD

Comparta dudas, retos y logros para avanzar como equipo.

Sobre el programa

Este frontend ligero está construido con HTML, CSS y JavaScript modular para facilitar futuras iteraciones. La integración a través de un servicio API dedicado permite cambiar la URL sin tocar las vistas.
